# -README.md
泰坦尼克号数据分析与可视化
项目概述
本项目旨在对泰坦尼克号乘客数据进行全面分析，使用决策树和随机森林模型预测乘客的生存情况，并通过多种可视化方式展示数据特征、模型性能和预测结果。
功能特性
数据加载与预处理：支持加载真实数据集和生成模拟数据集，处理缺失值，创建新特征。
数据探索可视化：展示生存状态、性别、舱位等级、年龄、票价和家庭大小与生存状态的分布关系。
模型构建与评估：使用决策树和随机森林模型进行训练和预测，评估模型性能。
性能对比可视化：对比决策树和随机森林模型的准确率、精确率、召回率、F1 分数和 ROC-AUC，并展示混淆矩阵。
特征重要性对比可视化：展示决策树和随机森林模型中各特征的重要性排序，并对比前 5 个重要特征。
决策树与随机森林结构可视化：可视化决策树和随机森林中部分决策树的结构。
预测结果对比可视化：对比决策树和随机森林模型的整体预测正确率，以及按性别和舱位等级的预测正确率。

依赖安装
确保你已经安装了以下 Python 库：
bash
pip install pandas numpy matplotlib seaborn scikit-learn

代码运行
将sum.py文件和Titanic_train.csv文件（如果使用真实数据集）放在同一目录下。
打开终端，导航到项目目录。

运行以下命令：
bash
python sum.py

文件说明
sum.py：包含数据加载、预处理、模型构建、评估和可视化的主要代码。
titanic_train.csv：泰坦尼克号乘客的训练数据集（可选）。
